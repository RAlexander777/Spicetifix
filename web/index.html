❯ SPICETIFIX v0.2.0 (Spicetifix / Tauri Web Engine)

Spicetifix es un centro de control terminal de diseño minimalista e instalador automatizado para Spotify y Spicetify en Windows.

Autor / Creador: RAlexander777
Licencia: MIT Open Source License
Motor: Python 3.11 + Win32 Media API + Web Engine
Repositorio GitHub: RAlexander777/Spicetifix
Web de Spicetify: spicetify.app
Web de Spotify: spotify.com

Funcionalidades Principales:

  • Instalación paso a paso o en 1-clic de Spotify, Spicetify CLI, Marketplace y Temas.
  • Gestor visual de extensiones con interruptores en vivo.
  • Reproductor integrado de Spotify mediante llamadas nativas Win32 API.
  • Recuperación automatizada post-actualización de Spotify (spicetify restore backup apply).
  • Respaldo e Importación completa de configuraciones y temas (.zip).
